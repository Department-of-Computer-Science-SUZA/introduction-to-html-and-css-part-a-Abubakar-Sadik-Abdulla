Role of the Head Section

The head section holds metadata, such as title, charset, and links to stylesheets, controlling the webpage’s setup and display.

Importance of Semantic Tags

Semantic tags like section, article, and footer organize content clearly, boost accessibility, and improve SEO.

Part 2: Exploring the DOM

The HTML file is static code, while the DOM is the browser’s dynamic, tree-like structure of the webpage, editable via JavaScript.

Modifying DOM nodes in developer tools changes the live page temporarily, not the HTML file, and resets on refresh.

DOM Structure

This text is nested in a div, a child of the body in the DOM hierarchy.

Part 4: CSS Basics

Inline CSS uses the style attribute for single elements.

Internal CSS is in a style tag in the head for one file.

External CSS is a separate .css file, reusable across pages.

Style conflicts are resolved by specificity (ID > class > element) or the last rule in the cascade.

Part 5: Responsive Web

Site Navigation

Responsive design adapts websites to various devices, ensuring usability and aesthetics on all screen sizes.

Mobile-first design focuses on smaller screens first, optimizing performance before scaling to larger displays.
